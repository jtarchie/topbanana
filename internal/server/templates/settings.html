Settings

Custom domains

One hostname per line. Point each domain at this server with a CNAME/A record. Static HTML is cached at the edge for 5 minutes; /api/* calls bypass the cache so any KV state stays live.

Dynamic features

{{ if .FunctionsByTmpl }}

This site was built from a template that already includes server-side handlers. Dynamic features are always on. (/api/* routes through the sandbox; the agent can author functions.)

{{ else }}

Server-side handlers let the agent build forms with persistence, counters, RSVPs, tiny shops — anything that needs state. Turning this on gives the agent the function-authoring tools on the next edit; turning it off makes /api/* return 404 but keeps any existing handler files.

{{ end }}

Public API

By default, /api/* POSTs only accept requests whose Origin or Referer matches this site — the Rails-style supplemental check that blocks cross-origin form spam. Turn this on if you want other sites (webhooks, public JSON endpoints, third-party integrations) to be able to POST to your handlers.

Danger zone

Deletes all files, snapshots, and stored data for this app. Cannot be undone.
