# Birthday announcement

## Purpose
A festive single-page announcement for a birthday or party — the agent fills in the celebrant's name, date, time, location, and RSVP method from the user's prompt. Static; no forms, no persistence.

## What ships
- `skeleton/index.html` — placeholder page with all the fields the agent should replace.

## Checks
- `index.html` must contain `<h1` — every announcement needs a big headline with the celebrant's name or party title.
