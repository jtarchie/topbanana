# Link in bio

## Purpose
A centered profile page — avatar, name, one-line bio, vertical stack of full-width link buttons. The Linktree replacement.

## What ships
- `skeleton/index.html` — single-column mobile-first layout.

## Checks
- `index.html` must contain `<h1` and at least one `<a ` — a name and at least one link is the bare minimum.

## Gotchas
The prompt nudges the agent toward expressive themes (`cupcake`, `valentine`, `synthwave`) and a gradient background. Without this guidance the agent tends to produce a flat white page that doesn't read as a personal profile.
